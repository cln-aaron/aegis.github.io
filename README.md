# AEGIS·AI

Website for **AEGIS·AI** — security operations for AI, by Hesed & Emet.

Static site, no build step:

| File | Purpose |
|---|---|
| `index.html` | Marketing site with animated product preview |
| `login.html` | Sign-in / demo-environment entry |
| `dashboard.html` | Interactive MVP console for customer demos |
| `styles.css` / `script.js` | Shared landing-page assets |

Open `index.html` locally or serve the directory. Deploys as-is via GitHub Pages.
The dashboard is fully interactive (synthetic demo data) — drive it live in a demo.
